Native Dialog + Custom Banner

The browser's native "Leave site?" prompt still fires. But inside that same beforeunload event we also run our own logic — here, revealing a banner on the page. If the user clicks "Cancel" / stays, they're left looking at your message.

Try to reload or close the tab: the native dialog appears, and the banner slides in behind it. The native dialog's own look/text can't be changed — but your page can.
